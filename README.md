# Fotos Geometrix

Repositorio **público** de imágenes de producto GEOMETRIX, usado únicamente para
hospedar las fotos y obtener **URLs directas** que se pegan en los Excel de carga
de Amazon (columnas de imagen). Amazon descarga cada imagen desde su URL y la
re-hospeda en sus servidores.

Proyecto independiente — no tiene relación con la plataforma interna.

## Estructura
```
<producto>/<listado>/01.jpg, 02.jpg, ...   (fotos en el orden de Amazon)
```

## Cómo se usa la URL en el Excel
```
https://raw.githubusercontent.com/inovacharlie/Fotos-Geometrix/main/<ruta>/01.jpg
```
